A full-stack web application where users can create, assign, and manage tasks with role-based access (Admin / Member).

## 📌 Features

### 🔐 Authentication
- User Signup & Login
- JWT-based authentication
- Secure password hashing using bcrypt

### 👥 Role-Based Access
- **Admin**
  - Create tasks
  - Assign tasks to users
  - Edit & delete tasks
  - View all tasks

- **Member**
  - View only assigned tasks
  - Update task status (Todo → In Progress → Done)

---

### 📋 Task Management
- Create tasks
- Assign tasks to users
- Update task status
- Edit task title
- Delete tasks

---

### 📊 Dashboard
- Kanban-style board:
  - Todo
  - In Progress
  - Done
- Clean glass UI with Tailwind CSS
